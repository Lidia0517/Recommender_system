# Recommender_system

# Movie Recommender System & Revenue Prediction


TMDB 5000 ֆիլմերի տվյալների բազայի հիման վրա՝ այս նախագիծն ուսումնասիրում է ֆիլմերի առաջարկման (movie recommendation) երկու մոտեցում և կիրառում է KNN մոդելը՝ ֆիլմի եկամուտը կանխատեսելու համար։

---

## Project Structure & Steps

### 1. Data Cleaning & Preprocessing
* **Տվյալների միավորում:** Ֆիլմերի (`movie`) և դերասանական/նկարահանող կազմի (`credits`) տվյալները միավորվել են մեկ միասնական DataFrame-ի մեջ՝ օգտագործելով `title` սյունակը։
* **Բացակայող արժեքների մշակում:** Հեռացվել են այն տողերը, որոնք չունեին համառոտագիր (`overview`), իսկ մնացած դատարկ դաշտերը լրացվել են «Unknown» արժեքով:
* **Տեքստի փոխակերպում:** Տեքստային սյունակները (`genres`, `keywords`, `cast`, `crew`) `ast.literal_eval` մոդուլի միջոցով վերածվել են օգտագործելի ցուցակների (lists):
* **Հատկանիշների առանձնացում:** Յուրաքանչյուր ֆիլմից առանձնացվել են առաջին 3 գլխավոր դերասանները և ռեժիսորը:
* **Տոկենիզացիայի օպտիմալացում:** Հեռացվել են բացատները բազմաբառ անուններից (օրինակ՝ «Tom Cruise» $\rightarrow$ «TomCruise»), որպեսզի դրանք դիտարկվեն որպես մեկ միասնական token:
* **Վիզուալիզացիա:** Վիզուալիզացվել են եկամտի միտումները ժամանակի ընթացքում՝ ներառելով 1935–1945 թվականների ժամանակահատվածի ավելի մանրամասն դիտարկումը:

### 2. Demographic Filtering
Այս մոդուլն առաջարկում է ֆիլմեր՝ հիմնվելով դրանց ընդհանուր ճանաչության և որակի վրա: Միևնույն առաջարկները ցուցադրվում են բոլոր օգտատերերին՝ անկախ անհատական ճաշակից։
Օգտագործվել է IMDB-ի կշռված վարկանիշի (**Weighted Rating**) բանաձևը.

$$Weighted\ Rating = \left(\frac{v}{v + m} \cdot R\right) + \left(\frac{m}{m + v} \cdot C\right)$$

Որտեղ՝
* $v$ - ֆիլմի ստացած ձայների քանակն է (`vote_count`)
* $R$ - ֆիլմի միջին գնահատականն է (`vote_average`)
* $m$ - պահանջվող ձայների նվազագույն շեմն է (ձայների քանակի 90-րդ percentile)
* $C$ - բոլոր ֆիլմերի միջին գնահատականն է

Ֆիլտրվել են այն ֆիլմերը, որոնք համապատասխանում են ձայների նվազագույն շեմին, և դասակարգվել են ըստ score-ի՝ **«Լավագույն ֆիլմեր» (Top Movies)** աղյուսակը ստանալու համար:

### 3. Content-Based Filtering
Առաջարկում է տվյալ ֆիլմին նման այլ ֆիլմեր՝ հիմնվելով content based հատկանիշների վրա։ 

* **Թեգերի ստեղծում:** Միավորվել են `overview`, `genres`, `keywords`, `cast` և `crew` սյունակները մեկ միասնական `tags` սյունակի մեջ:
* **Stemming:** Կիրառվել է NLTK գրադարանի `PorterStemmer`-ը՝ բառերի հիմքերը նորմալացնելու համար:
* **Վեկտորացում:** `tags` սյունակը վեկտորացվել է `CountVectorizer`-ի օգնությամբ (վերցվել են թոփ 5000 բառերը, իսկ անգլերենի stop words-ը հեռացվել են):
* **Նմանության հաշվարկ:** Հաշվարկվել է զույգ առ զույգ կոսինուսային նմանությունը (**Cosine Similarity**) բոլոր ֆիլմերի վեկտորների միջև:
* **Ֆունկցիա:** Ստեղծվել է `recommend(movie)` ֆունկցիա, որը վերադարձնում է տվյալ ֆիլմին ամենանման թոփ 10 ֆիլմերը:

### 4. Եկամտի կանխատեսում KNN-ի միջոցով (Revenue Prediction)
* **Թիրախային փոփոխական (Target):** `revenue` (եկամուտ), որը բաժանվել է երեք դասի՝ **Ցածր (Low)**, **Միջին (Medium)**, **Բարձր (High)**:
* **Հատկանիշներ (Features):** `budget`, `popularity`, `runtime`, `vote_average`, `vote_count`, `release_year`, `genre_count`, `cast_size`, ինչպես նաև `top-genres`:
* **Մոդելավորում:** Տվյալները ստանդարտացվել են `StandardScaler`-ի միջոցով և բաժանվել training ու test սեթերի (80/20 հարաբերակցությամբ)։ Օպտիմալ հիպերպարամետրն է՝ **k = 6**:
* **Արդյունքներ:** Մոդելը գնահատվել է accuracy, precision, recall և confusion matrix-ի միջոցով: Այն բավականին լավ է տարբերակում «Բարձր» (High) և «Ցածր» (Low) եկամտաբերություն ունեցող ֆիլմերը (հստակությունը՝ ~0.73–0.75), սակայն դժվարանում է «Միջին» (Medium) դասի դեպքում (հստակությունը՝ ~0.50):

## Dataset

**Աղբյուր:** [Kaggle - TMDB Movie Metadata](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)

Նախագծում օգտագործվել են հետևյալ երկու CSV ֆայլերը.

### 1. `tmdb_5000_movies.csv` (Ֆիլմերի բնութագրեր)

| Սյունակ | Նկարագրություն |
| :--- | :--- |
| `budget` | Բյուջեն, որով նկարահանվել է ֆիլմը |
| `genres` | Ֆիլմի ժանրերը (Action, Comedy, Thriller և այլն) |
| `homepage` | Ֆիլմի պաշտոնական կայքի հղումը |
| `id` | Ֆիլմի եզակի ID-ն |
| `keywords` | Ֆիլմին առնչվող բանալի բառերը կամ թեգերը |
| `original_language` | Այն լեզուն, որով նկարահանվել է ֆիլմը |
| `original_title` | Ֆիլմի սկզբնական անվանումը՝ նախքան թարգմանվելը |
| `overview` | Ֆիլմի համառոտ նկարագրությունը |
| `popularity` | Ֆիլմի հանրաճանաչության աստիճանը |
| `production_companies`| Արտադրող ընկերությունները |
| `production_countries`| Երկրները, որտեղ նկարահանվել է ֆիլմը |
| `release_date` | Ֆիլմի թողարկման ամսաթիվը |
| `revenue` | Եկամուտն համաշխարհային կինովարձույթում |
| `runtime` | Ֆիլմի տևողությունը՝ արտահայտված րոպեներով |
| `status` | Ֆիլմի կարգավիճակը ("Released" կամ "Rumored") |
| `tagline` | Ֆիլմի կարգախոսը (սլոգանը) |
| `title` | Ֆիլմի անվանումը |
| `vote_average` | Ֆիլմի ստացած միջին գնահատականը |
| `vote_count` | Ֆիլմի ստացած ձայների ընդհանուր քանակը |

### 2. `tmdb_5000_credits.csv` (Դերասանական և նկարահանող կազմ)

* `movie_id` - Ֆիլմի ID-ն (համապատասխանում է առաջին դատասեթի `id` սյունակին)
* `cast` - Գլխավոր դերասանների անունները
* `crew` - Ռեժիսորի, մոնտաժողի, կոմպոզիտորի, սցենարիստի և այլ անդամների տվյալները
